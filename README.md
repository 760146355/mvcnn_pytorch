# mvcnn_pytorch
PyTorch code for MVCNN  
Code is tested on Python 3.6 and PyTorch 0.4.1

Command for training:  
```python train_mvcnn.py -name mvcnn -num_models 1000 -weight_decay 0.001 -num_views 12 -cnn_name vgg11```
